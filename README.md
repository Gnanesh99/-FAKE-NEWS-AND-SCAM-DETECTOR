🧠 Student Mental Health Risk Predictor

A machine learning-based web application that predicts the mental health risk level of students based on input factors. This project aims to provide early insights into potential mental health concerns using data-driven analysis.

🚀 Features

Predicts mental health risk level (Low / Medium / High)

User-friendly interface built with Streamlit

Fast and real-time predictions

Simple input-based assessment system

🛠️ Tech Stack

Python

Machine Learning (Scikit-learn / Pandas / NumPy)

Streamlit (Frontend + Deployment)

📊 How It Works

User inputs relevant details (stress, sleep, lifestyle, etc.)

The trained ML model processes the data

The system predicts the mental health risk level

Output is displayed instantly on the web interface

🌐 Deployment

The application is successfully deployed and accessible online.
(This marks my first deployed ML project 🚀)

▶️ Run Locally
# Clone the repository
git clone <your-repo-link>

# Navigate to project folder
cd student-mental-health-risk-predictor

# Install dependencies
pip install -r requirements.txt

# Run the app
